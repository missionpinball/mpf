This folder contains subfolders for different pinball machines. Each machine's
folder has further subfolders for configuration files, machine-specific code,
light shows, DMD files, audio, etc.

There are many more machine configurations in the Mission Pinball GitHub page:
https://github.com/missionpinball/

If you're new to MPF, start with the tutorial at
http://missionpinball.com/docs/tutorial.

Note that no copyrighted information or intellectual property is included in any
of these files. For example, the Demo Man sample machine is not Demolition Man
per se, rather it is new code unrelated to Demolition Man which can run on a
Demolition Man pinball machine.

demo_man
========
Williams Demolition Man, the current dev machine for the Mission Pinball
Framework project.

new_machine_template
====================
A template folder structure you can use as the basis for your own machines.

tutorial
========
Contains the companion configuration files for each step in our step-by-step
tutorial.
